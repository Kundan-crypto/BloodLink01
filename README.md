## 🚀 Overview

A modern **Blood Donation Finder App** built using **Flutter** with a premium UI and smooth animations.  
This app helps users **find nearby blood donors**, **request blood in emergencies**, and **connect with lifesavers** ❤️

---

## ✨ Features

- 🔍 Find nearby blood donors
- 🩸 Request blood instantly
- 👤 Donor profile with details
- 🗺️ Location-based donor search
- 🔔 Real-time notifications (UI)
- 🌗 Dark & Light mode support
- ⚡ Smooth animations & transitions
- 🎨 Modern glassmorphism UI

## ⚙️ Getting Started

### 🔧 Prerequisites

- Flutter SDK installed
- Android Studio / VS Code
- Emulator or real device

---

### ▶️ Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/Blood-Donation-finder-App.git

# Navigate to project
cd Blood-Donation-finder-App

# Install dependencies
flutter pub get

# Run the app
flutter run
